# Fakts directory

This directory contains the fakts that are used for establishing
a central autoconfiguration of connected apps. (See [fakts](https://github.com/jhnnsrs/fakts) for more information)

It contains two subdirectories that establish the routing 
app specific configuration and a simple yaml templates to the app
configuration. (See [fakts](https://github.com/jhnnsrs/fakts) for more information) ONGOINg
